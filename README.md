# Algo Study

## 규칙

- 실전 코딩 테스트 환경과 동일 환경 라이브 코딩이 목적
- 미팅 시 2시간 동안 네 문제(실버, 골드)에 대해 논의없이 각자 라이브 코딩 진행
- 이후 풀이 방법, 시간복잡도에 대한 토의 진행
- 이 레포지토리에는 `개인 폴더`에 `각 주차 하위 폴더`를 만들어 `제한시간까지 풀이를 진행한 것까지의 코드`, `이후 개선된 코드`, 그에 대한 설명이 들어 있는 `리드미`를 저장한다.
